# YOLOv7_c_inference
An example of inferencing YOLOv7 by using opencv and onnxruntime on c.
This code is tested on windows.
## How to obtain yolov7.onnx?
* Please Generate the model file from the official implement of YOLOv7.
https://github.com/WongKinYiu/yolov7
## Dependencies
### opencv
* Import the opencv include directory.
* Link to opencvworldxxx.lib.
* Add opencvworldxxx.dll into environment variable or into the execution directory.
### onnxruntime_gpu
* Install with nuget.
